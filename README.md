# AION — Age Intelligence Engine

Пародийный «премиальный ИИ», который якобы определяет точный возраст по дате
рождения, а на деле считает возраст обычным скриптом и выдаёт приколы в стиле
разных нейросетей (GPT, Claude, Gemini, Grok, DeepSeek, Алиса, GigaChat и др.).

Промо-проект для вирусных Reels — [@art1uca](https://instagram.com/art1uca).

## Стек

Чистый статический сайт: HTML + CSS + ванильный JS. Никаких фреймворков,
бандлеров и шага сборки. Единственная внешняя зависимость — Google Fonts по CDN.

```
index.html    точка входа (подключает styles.css, models.js, app.js)
styles.css    кинематографичная «тёмно-кофейная» тема
models.js     каталог моделей и тексты приколов
app.js        логика и стейт-машина экранов
favicon.svg   иконка ◑ в фирменной палитре
og-image.png  обложка для шеринга (1200×630)
_redirects    редирект www → без www (Cloudflare Pages)
```

## Локальный запуск

Любой статический сервер из корня репозитория:

```bash
# Python (есть почти везде)
python3 -m http.server 8000

# или Node
npx serve .
```

Затем открыть <http://localhost:8000>.

> Открывать `index.html` напрямую через `file://` не рекомендуется — браузеры
> могут блокировать загрузку соседних файлов. Используйте локальный сервер.

## Деплой

Статикой, без шага сборки. Настройки Cloudflare Pages:

- Framework preset: **None / Static**
- Build command: *(пусто)*
- Build output directory: **`/`** (корень репозитория)

Кастомные домены: `aion.art1uca.com` (основной) и `www.aion.art1uca.com`
(редиректит на основной через `_redirects`).

## Скрытая «панель Режиссёр»

Маленькая точка в правом нижнем углу открывает панель управления: можно
форсировать исход, скорость загрузки и режим (успех/ошибка) для нужной модели —
удобно при записи Reels.
